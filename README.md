## To run backend

> ``` cd backend/python

> ```sudo apt install python3```

> ```sudo apt install python3-pip```

Create a virtual environment

> ```pip3 install --user virtualenv```

> ```virtualenv venv```

> ```source venv/bin/activate```

Install dependencies

> ```pip3 install -r requirements.txt```


## To run frontend

> ``` cd frontend```

> ``` yarn install```

> ``` yarn start```
